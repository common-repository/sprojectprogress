Create a widget with progress bars for your projects.

== Description ==

This Wordpress plugin allows you to create a widget with progress bars for your projects. You can choose between several designs and select your own colors for them.

= Upcoming =

* More appearances
* More settings to costumize the progress bars

== Installation ==

1. Upload all extracted files of the sprojectprogress.zip into your wordpress plugin directory (e.g. /wp-content/plugins/sprojectprogress/).
2. Activate the plugin in the admin control panel.
3. Done!
